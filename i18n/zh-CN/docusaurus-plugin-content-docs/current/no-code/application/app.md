在Steedos平台上，应用是一系列互联的组件，包括对象、微页面、报表、仪表板和其他业务流程，共同支持一个业务功能。此平台让您能够通过图形界面快速构建、自定义和部署专业级应用，无需深入编程知识。


## 新建应用

1. 登录 Steedos 低代码平台。
2. 在设置界面上，找到并点击“应用程序”。
3. 选择“新建”。
4. 输入应用名称、描述，并选择一个适合的图标。

## 设定导航菜单

应用导航菜单可以显示在顶部或左侧。显示在左侧的菜单可以设置分组。

导航菜单由选项卡构成，您必须先创建选项卡，然后再把选项卡添加到应用中。
